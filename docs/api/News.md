# News

## Update News

| Property | Value |
|:---------|:------|
| URL | /pending |
| Method | PUT |
| Post Data Form | JSON |
| Return Data Form | JSON |

### Post Data

| Property | Type |
|:---------|:------|
| url | url |
| source | string |
| title | string |
| abstract | text |
| time | date |
| status | string |

**All properties are optional. The excluded properties means it does not need to be changed.**
